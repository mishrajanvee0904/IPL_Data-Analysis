# IPL_Data-Analysis
# IPL Data Analysis (2008–2024)

## About
End-to-end IPL data analysis answering 17 business questions 
using Python (Pandas, Matplotlib, Seaborn)

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
Download dataset from Kaggle:
https://www.kaggle.com/datasets/patrickb1912/ipl-complete-dataset-20082020

## Questions Answered
1. Does winning toss win the match?
2. Top 10 run scorers of all time
3. Top 10 wicket takers of all time
4. Batting first vs chasing win rate by team
5. Batting vs bowling friendly venues
6. How many matches are close finishes?
7. Which batsman hits most boundaries per innings?
8. Which bowlers have most dot balls?
9. Most consistent team overall
10. Has scoring increased season by season?
11. Best death over batsmen (SR in overs 17-20)
12. Does powerplay score predict final total?
13. Best powerplay bowlers by economy
14. Best vs worst death over bowlers
15. What score is safe to defend?
16. Do extras cost you the match?
17. Which dismissal type is most common per phase?
